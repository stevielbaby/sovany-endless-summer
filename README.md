# Sovány: Endless Summer

Premium glass bottle campaign landing page for CEO review.

## About

This is an internal marketing proposal for the **Sovány: Endless Summer** campaign - a limited-edition glass bottle collection that celebrates laid-back summer moments and elevates the brand's organic sparkling water into collectible luxury.

## Campaign Vision

- **Tagline**: Savor the Spirit of an Endless Summer
- **Format**: Limited-edition glass bottles with collectible sleeve art
- **Strategy**: Curated drops at premium venues that embody summer's golden hour aesthetic
- **Goal**: Transform everyday refreshment into shareable lifestyle moments

## Features

- Responsive React + Tailwind CSS design
- Motion animations for smooth scroll interactions
- Brand-aligned warm golden color palette
- Authentic Sovány product photography integration
- Campaign story focused on organic storytelling and brand elevation

## Tech Stack

- **Framework**: React 18.3.1
- **Styling**: Tailwind CSS v4
- **Animations**: Motion (Framer Motion)
- **Build Tool**: Vite
- **Package Manager**: pnpm

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Brand Values

- **Honest**: Real fruit, pure taste, straightforward and real
- **Laid Back**: Savoring moments, encouraging play & relaxation  
- **Sophisticated**: Fresh take on classic style with a certain panache

## Campaign Highlights

- 4 distinct flavors: Raspberry, Tangerine, Green Apple, Pineapple
- 12% real fruit juice, USDA Certified Organic
- Only brand that's both Gluten Free AND Organic
- Glass bottles designed to be photographed, shared, and kept
